# 主要功能
程序主要模拟了网络中加密传输的过程，
包括加密和解密、传输的过程、
重点使用了rsa aes两个加密算法。

# 环境
电脑环境 window7 x64
需要python2.7
安装包rsa, Cryptodome
本实例在windows下运行

# demo
启动工程，先启动server.py
然后启动client.py
根据提示，进行一系列操作
服务端监听本地端口号2409
客户端发送数据到2409
进行加密数据交互

# 支持的验证
可以支持中文、英文的加密
支持从文件中读取数据，也可以手写，
可以选择加密方式。
注意，读取文件遗留一个问题，多行文本，没有转义，目前只能读取一行数据
